Searching

Searches are not case sensitve. SMITH is exactly the same as smith.

For more results use less text for example to find parcel # "123-123-34-234" enter "123-123-34"
and all parcels with that partial number will be displayed.
Or, if looking for an address say "123 Main ST. S" you could enter "123 Main" or "Main St".

The Toolbar

Some of these buttons may not appear on your toolbar.

  • Zoom In: Draw a rectangle on the screen to zoom to that view
  • Zoom Out: Draw a rectangle on the screen to zoom to that view
  • Zoom Full Extent: Zooms back to the initial view of the map
  • Zoom Prev: Zooms to the last veiw
  • Zoom Next: Zooms to the next view
  • Pan: Click the map and drag to move around the map without zooming
  • Identify: Click the map to view attributes of objects
  • HyperLink: Click the map to view a new page about the clicked object
  • Print: Get a pdf of the map view and attributes of any identified objects
  • Clear Selection: Remove all selected objects from the map
  • Select by Rectangle: Draw a rectangle on the screen to select items, you will then see a window with the results and an option to download the results.
  • Measure (point): Show the XY coordinates of a point on the map
  • Measure (line): Show the length of a line drawn on the map
  • Measure (area): Show the area and perimiter of a polygon drawn on the map.
