Prerequisites

  1. Java you must have JDK version 1.5 or greater installed and the following java utilities and libraries download

    • apache ant download
    • htmlcompressor.jar download
    • closure.jar download
    • yuicompressor.jar download

  2. Python version 2.6 or greater installed and the following modules download

    • python-setuptools optional but recomended download
    • pip (alternative to python-setuptools) download

    • jinja2 python module

      Install with using easy_install (python-setuptools) or pip
    easy_install Jinja2
    pip install Jinja2

  3. Mercurial download

Request Access to the agdjsapi repository

  1. create an account at bitbucket
  2. email mbratcher@atlasgeodata.com with your bitbucket username to gain read only access to the repository
